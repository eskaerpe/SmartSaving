# SmartSave 🎯

A modern, AI-powered web-based savings planner that helps you achieve your financial goals with ease! 🌟

## 📋 About SmartSave

SmartSave is a web application that simplifies the process of saving money. Whether you're saving for a new gadget 📱, a dream vacation 🏖️, or an emergency fund 💰, SmartSave breaks down your savings goals into manageable daily, weekly, and monthly contributions.

## 🚀 Features

- 💰 **Custom Savings Goals**: Enter any amount you want to save
- ⏳ **Flexible Duration**: Choose from 1 to 24 months
- 🔄 **Real-time Calculations**: Instant updates as you type
- 📱 **Responsive Design**: Works perfectly on any device
- 🛡️ **Input Validation**: Smart error handling
- 🌐 **Modern UI**: Clean and intuitive interface

## 📱 How to Use

1. Enter your savings goal amount in Rupiah (IDR)
2. Select your desired saving duration (1-24 months)
3. Click "Calculate Plan" or press Enter
4. View your daily, weekly, and monthly savings targets
